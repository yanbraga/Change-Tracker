
Privacy Policy for SF Change Tracker

Last updated: March 2026

SF Change Tracker does not collect, store, transmit, or share any personal data or user information with third parties.

DATA USAGE
The extension accesses the following data solely to provide its core functionality:

Salesforce session cookie (sid): used exclusively to authenticate requests to the Salesforce REST API of the user's own org. This value is never stored, logged, or transmitted outside the user's browser.

SetupAuditTrail records: fetched directly from the user's Salesforce org and displayed within the extension popup. This data is never stored, cached, or sent to any external server.

User preferences (filter toggles): saved locally in the browser using Chrome's storage API. This data never leaves the user's device.

EXTERNAL SERVICES
This extension does not communicate with any external servers, analytics services, or third-party APIs. All data processing happens entirely within the user's browser.
